# remote-heartbeat-research-angelofurheart
## Overview

This project explores the possibility of detecting human heartbeat remotely using wireless technologies, signal processing, and machine learning.

## Goals

- Measure heartbeat using MAX30102
- Collect motion data using LD2410C radar
- Investigate Wi-Fi based sensing techniques
- Apply signal processing and FFT analysis
- Explore machine learning approaches
- Compare contact and contactless measurements

## Hardware

- ESP32
- MAX30102
- LD2410C
- RTL8812AU
- Breadboard and jumper wires

## Technologies

- Python
- NumPy
- SciPy
- Matplotlib
